The concat method will allow you to append several variables to 1 variable.

The slice method will display the parameters given. You can start anywhere in the string using its index.

The toUpperCase method will return its string value in uppercase letters.

The toSearch method will search for the parameters given and return the index of the first result.

The toString method will return an integer as a string.

The toprecision method will return a value of a number depending on the specified parameter if given. The number is 531.4598 and the parameter is 4.

The toFixed method is used to format a number with a specified number of decimal places and reutrn a string representation. the number is 531.4598

The valueof method will return the primitive value of an object, string, date, and number.
